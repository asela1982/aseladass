Developing this website...

This website was built using the learning acquired from the UT Data Analytics program. I purposely stayed away from using a template so I could expose myself more to HTML, CSS, Bootstrap, and JavaScript technologies. The visual inspiration to create this website was acquired from Scott Murray, the author of the book Interactive Data Visualization for the web, and of course, some of the d3 plots used in this site was based on the coding examples given on his amazing book aswell as from examples provided by Mike Bostock.
