About

Formed the Colombo Data-Driven Meetup group during the beginning of the year 2019. The objectives of forming the group were to enhance my communication skills; to have a platform to share my data-analytical knowledge; and through teaching to further improve my data-analytical skills.

The details section shows the present number of members in the group aswell as details of all the meetups I have conducted so far. As these information is fetched, real-time, from the meetup api, you might experience some latency in rendering the information.

The meetups are structured under two tracks:

Data Analyst track

The objective of this track is to introduce participants to a set of analytical skills needed to support the collecting, cleansing, transforming, visualizing, and analyzing of data using next-generation tools and platforms such as KNIME, Power BI, and Tableau.

  • Learn SQL Basics
  • Level Up SQL!
  • Know your NoSQL with MongoDB
  • Data Preparation and Analytics with KNIME
  • Data Storytelling with Tableau
  • Dashboarding with PowerBI

Data Scientist track

This track introduces attendees to carry out data analytical tasks programatically using Open-source, scientific libraries such as Pandas, Matplotlib, Seaborn, Scikit-learn, and Numpy in Python.

  • Introduction to the Bash Command line
  • Version Control with Git
  • Programming with Python - Basics
  • Programming with Python - Intermediate
  • Data Analysis with Pandas
  • Introdction to Machine Learning
  • Kaggle Competition
